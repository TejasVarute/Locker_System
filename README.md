# 🔐 Locker System

A GUI-based **Bank Locker Management System** built with **Python**, **CustomTkinter**, and **SQLite**. This application helps banks efficiently manage locker allocations with facial recognition-based authentication, categorized locker booking, and a modern, user-friendly interface.

---

## 🧰 Features

- 🧍‍♂️ Customer Information Management  
- 🧾 Locker Booking (Small / Medium / Large categories)  
- 🧠 Facial Recognition for authentication (OpenCV)  
- 🎨 Interactive CustomTkinter GUI  
- 📸 Photo, Signature, and ID Proof Uploads  
- 📊 Real-Time Locker Availability Display  
- 🗃️ SQLite3 for persistent data storage  
- 🖥️ Works on both 32-bit and 64-bit Windows systems  
- 🔒 Admin access for overview and management
